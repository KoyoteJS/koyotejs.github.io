Lightweight

						
Just a script tag and just 550 bytes (gzipped) away in the browser or $ npm install koyotejs away in node.

					

					

						
						
Open Source

						
KoyoteJS is a free to use on any platform. For personal or commercial use. So go ahead and use it!

					

					

						
						
Easy to Learn

						
Requires some programming knowledge, however KoyoteJS syntax is easy to learn and quickly!
